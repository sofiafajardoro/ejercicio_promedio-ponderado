Ejercicio Práctico – Promedio Ponderado del Curso:

Desarrollar un programa en JavaScript que permita calcular el promedio de un curso a partir de las notas de sus alumnos, aplicando correctamente ponderaciones diferenciadas. Con esta actividad practicarás el uso de ciclos (for), estructuras de control, operaciones aritméticas y la interacción con el usuario a través de entradas y salidas en el navegador.
